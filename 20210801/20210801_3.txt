3. Да се преобразуват от десетична в шестнадесетична бройна система
числата:
а) 312 б) 256 в) 123 г) 317

а) 312
312 / 16 = 19   a0 = 8
19  / 16 = 1    a1 = 3
1   / 16 = 0    a3 = 1
312 = 138(16)


б) 256
256 / 16 = 16   a0 = 0
16  / 16 = 1    a1 = 0
1   / 16 = 0    a2 = 1
256 = 100(16)


в) 123
123 / 16 = 7    a0 = 11(B)
7   / 16 = 0    a1 = 7
123 = 7B(16)


г) 317

19  / 16 = 1    a1 = 3
1   / 16 = 0    a2 = 1
317 = 13D(16)
